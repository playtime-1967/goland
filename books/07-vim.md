Command Line mode
Pressing the colon (:)
:q!  quit without saving
:x!  quit with saving
:set number  show line numbers
:20  navigate to line 20th.
:$   last line
dd   remove a line
u    ctrl+z
v    highlight mode
y    copy
d    cut
p    paste
o    enter new line
/text  search
n    find the next match.
